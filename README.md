# Discord Insights

member activity watcher in Discord

## Installing

**Python 3.10 is required**

```shell
$ git clone git@github.com:cashmere53/discord-insights.git
$ cd discord-insights
$ python -m pip install -U .
```

### for Development

```shell
$ git clone git@github.com:cashmere53/discord-insights.git
$ cd discord-insights
$ poetry install
```

## Quick Start

```shell
$ python -m dinsights
```
